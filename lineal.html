Estructuras de Control

Las estructuras de control, como su nombre lo indica, son estructuras que controlan las secuencias lógicas de un algoritmo para cumplir con cierta condición, evaluar un error o repetir varios procesos.

Lineal

Permiten que las instrucciones se ejecuten de manera secuencias, es decir, una tras otra, en el orden en el que están escritas.

If

Permite que un programa ejecute un bloque de código solo si se cumple una condición específica. Si la condición es verdadera, se ejecutan las instrucciones dentro del bloque, de lo contrario, se omiten.

If Anidado

Permite evaluar múltiples condiciones en un programa. Un bloque if puede contener otro if dentro de su bloque, lo que permite realizar decisiones más complejas. Si la primera condición es verdadera, se ejecuta su bloque; si es falsa, se evalúa la siguiente condición anidada.

If/Else

Permite que un programa tome decisiones basadas en condiciones. Si la condición en el if es verdadera, se ejecuta un bloque de código; si es falsa, se ejecuta el bloque del else.

While

Permite ejecutar un bloque de código repetidamente mientras una condición específica sea verdadera. Se evealúa la condición, si es verdadera, se ejecuta el bloque de instrucciones y luego se vuelve a evaluar la condición, este proceso se repite hasta que la condición se evalúa como falsa.

Do While

Permite ejecutar un bloque de código al menos una vez y luego continuar repitiéndolo mientras una condición sea verdadera. Primero ejecuta el bloque y solo despues ejecuta la condición.

Case

Permite ejecutar diferentes bloques de código según el valor de una variable. Compara el valor de una expresión con múltiples casos definidos.
